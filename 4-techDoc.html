Introduction

Here is an unexhaustive list of comparisons between the syntax of JavaScript and that of Python.

The JavaScript version is assumed to be ES5+ and that of Python 3+.

Comment

In JavaScript, source code comments can be inserted easily as either single-line or multi-line comments.

  • Single-line: // this is a JS single-line comment
  • Multi-line: /* this is a JS
       multi-line
       comment black
    */

In Python, there is native support for inserting single-line comments. Python, however, does not have a dedicated syntax for block comments -- at least not in the way that JavaScript does. While there is a workaround, one should be aware of its potential pitfalls as discussed in this blog post.

  • Single-line: # this is a Python single-line comment
  • Multi-line workaround (potential pitfalls): """
    this is a workaround for block comments in Python
    made using a multi-line string constant
    """

Variable Declaration

In JavaScript, to initialize a variable, one must declare it using the var keyword (let or const would also work if using ES6).

var a = "hello";

In Python, no special keyword is required for declaring a variable.

a = "hello"

Print

  • Javascript: console.log()
  • Python*: print()

*: Python 3 uses the syntax print("something"), whereas Python 2 uses print "something".

Boolean

The Boolean values in Javascript and Python differ in capitalization.

  • Javascript: true; false
  • Python: True; False

The Boolean operators in Javascript and Python also differ.

  • Javascript: &&; ||
  • Python*: and; or

*: For numpy Boolean arrays, use & and | instead.

Function

JavaScript and Python use different syntax for declaring functions.

  • Javascript: function checkSign(num) {
        return num>0 ? "positivie" : num == 0 ? "zero" : "negative";
    }
  • Python: def checkSign(num):
        if num>0:
            return "positive"
        elif num==0:
            return "zero"
        else:
            return "negative"

If no explicit return value is specified, a function returns:

  • Javascript: undefined
  • Python: None

In addition, in Python, it is possible for a single function to return multiple values.
